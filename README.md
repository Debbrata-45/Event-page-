# 🎓 College Event Registration System

A full-stack web application designed to manage and streamline college event registrations efficiently.

## 🚀 Features
- Student Registration & Login System  
- Admin Login Dashboard  
- Event Registration for multiple events  
- Input validation (Email, Phone, Required fields)  
- Duplicate registration prevention  
- Admin panel to view all students & event participants  

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB

## 📊 Functionality
- Students can register using name, email, phone, course, and password  
- Students can log in using email or phone number  
- Students can explore events and register with branch, semester, and KU ID  
- Admin can view all registered students and event participation data in tables
